EDD Algorithm

Step 1. Arrange the jobs in EDD order and assume this as set E. Let set J be empty.

Step 2. Let j be first element in EDD . If adding j to L is feasible , union J and j and delete j from E.

Step 3. Using max-heap to identify the longest job, among the first K jobs in the sequence. Remove this job from J. Revise the completion times of the jobs remaining E and return to step 2.
